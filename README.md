# TFM_Altadis_ABM-DLM

Este Trabajo Fin de Máster diseña y desarrolla un sistema de inteligencia de negocio aplicado a Altadis para apoyar la optimización de su estrategia de distribución al mercado. El proyecto parte de datos operativos anonimizados sobre ventas, entregas, productos, rutas, puntos de venta e incidencias de disponibilidad, que se depuran, estructuran y enriquecen con variables territoriales, meteorológicas y de calendario. La metodología combina procesos ETL en RStudio, modelado relacional en Microsoft Power BI y técnicas de Machine Learning para predicción de demanda, segmentación de estancos y clasificación del riesgo de rotura de stock. Como resultado, se obtiene una prueba de concepto funcional compuesta por modelos analíticos y un cuadro de mando ejecutivo interactivo. La principal conclusión es que, pese a las limitaciones del dato histórico, es posible transformar registros operativos heterogéneos en información útil, trazable y accionable para la toma de decisiones comerciales y logísticas.
